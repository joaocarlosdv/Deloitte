Para executar o sistema no Visual Studio, siga os seguintes passos:

1. Descompacte o arquivo Teste.Zip em uma pasta;
2. Abra o Visual Studio;
3. Clique na opção a direita "Abrir um projeto ou uma solução";
4. Localize o arquivo "Deloitte.sln" dentro da pasta Deloite;
5. Localize o arquivo "appsettings.json" que encontra-se dentro da pasta "WebApp\Deloitte";
5. Altere a string de conexão apontando para sua base de dados;
6. Localize o arquivo "script.sql" dentro do teste.zip;
7. Execute o script em sua base de dados para criar a tabela;
8. No visual studio execute a aplicação web Deloitte para rodar a aplicação;
